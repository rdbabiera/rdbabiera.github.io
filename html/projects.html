chIRC, a simple IRC Client

chIRC is an IRC server built according to RFC 1459 with limited functionalty according to Assignments 1 and 4 here. You can connect to the server using any IRC client, and assuming port-forwarding other users will be able to connect to the server as well. The source code is on my GitHub page.


The small demonstration above illustrates two users connecting to the server via telnet, and User One sending User Two a private message.
